# 更新日志

## [0.5.0] - 2026-05-15

### 新增
- 暗色主题切换（侧边栏底部 ☀/🌙 按钮，设置持久化）
- 端口转发协议选择器（TCP/UDP 可切换）
- WAN 卡片接口名称加粗显示

### 变更
- 自定义样式适配暗色/亮色双主题（侧边栏、卡片背景、内容区）
- 检测状态改走 pkexec 提权查询 iptables，结果更准确
- 启动时快速探测改用普通权限（不弹密码）
- 开始共享按钮移到 IP/掩码输入行右侧
- 各页面滚动条增加右侧内边距，避免覆盖卡片内容
- 蓝色标题文字暗色下自动变亮

### 修复
- 月亮图标渲染问题（改用 Advanced shaping）
- 暗色模式配色可见性（标题、说明文字）
- CI 触发频率（改为仅 tag 推送触发）

## [0.4.1] - 2026-05-15

### 修复
- 网络测试稳定性（轮询重试替代固定 sleep）
- Clippy warnings（dead_code 标注）
- 代码格式化对齐 CI 检查
- 侧边栏图标因宽度不同导致文字错位

## [0.4.0] - 2026-05-14

### 新增
- 代码架构重构：拆分 main.rs 为 app/types/theme/i18n/widgets/pages 模块
- 翻译系统外置：locales/*.json 编译时嵌入，支持中英文切换
- 每 LAN 共享独立 WAN 选择、独立开关按钮
- 外网接口池：每接口独立卡片展示 MAC/IP(v4/v6) 信息
- 颜色集中管理：colors.rs 统一所有颜色常量

### 变更
- 网络共享页布局重构：WAN/LAN 左右并排 → WAN 全宽顶部 + LAN 独立区域
- 就绪和刷新状态合并到同一行，宽度比例跟随按钮布局
- 所有卡片（WAN/LAN）去掉圆角，紧凑 padding
- 全局"开始共享"/"停止共享"按钮移除，改为每 LAN 卡片内独立控制
- "检测状态"按钮移到 WAN 池右上角

### 修复
- 页面内容过长时被窗口裁剪（添加 scrollable）
- 就绪/刷新框高度不一致（统一 button 包裹结构）
- 文本不渲染问题（Button::Text → Button::Secondary，去掉内层 container）
- LAN 卡片 WAN 选择自动排除已选 LAN 接口
- 测试时序问题（UDP recv 重试、stop_tx 忽略错误）

## [0.3.0] - 2026-05-11

### 新增
- 配置持久化：语言、关闭行为、转发规则自动保存到 ~/.config/conduit/config.json
- GitHub Actions CI：自动构建、lint、测试、release
- 单元测试 20 个：覆盖转发逻辑、状态机、序列化

### 变更
- 编译优化：LTO + strip + opt-level="z"，体积 38MB → 23MB
- Makefile 新增 dist/install-dist/uninstall-dist 目标
- .gitignore 补充 IDE、OS、运行时配置

### 修复
- 跨平台 Emoji 渲染：捆绑 Noto Sans Symbols 2 后备字体
- 启用 MSAA 抗锯齿，设最小窗口尺寸 800×600 防止崩溃
- 修复 ksni 0.3.4 API 兼容（Handle、TrayMethods、ToolTip）
- 修复窗口关闭事件订阅（close_requests → event::listen_with）
- 移除编译器 warnings
- 文档全部中文化
